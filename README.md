# Калькулятор

## Настройка проекта
```
npm install
```

### Компиляция и горячая перезагрузка для разработки
```
npm run serve
```

### Компилирует и минимизирует для производства
```
npm run build
```

### Запустите ваши тесты
```
npm run test
```

### Корпия и исправляет файлы
```
npm run lint
```
